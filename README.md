Buurtcampuskrant

Buurtcampuskrant is een digitale krant van de Buurtcampus.
Het doel van dit project is om nieuwsartikelen van de Buurtcampus overzichtelijk en toegankelijk online te tonen.

Het ontwerpprobleem was dat nieuwsartikelen duidelijk, gestructureerd en eenvoudig navigeerbaar moesten zijn voor bezoekers.

De oplossing is een simpele en duidelijke website waarbij bezoekers via een navigatiemenu makkelijk tussen verschillende pagina’s kunnen navigeren en nieuwsartikelen kunnen bekijken.

## Beschrijving
De Buurtcampuskrant is een website waarop nieuwsartikelen van de Buurtcampus worden weergegeven. Bezoekers kunnen via het navigatiemenu in de header eenvoudig tussen verschillende pagina’s navigeren waar artikelen gesorteerd worden weergegeven. Daarnaast kan een gebruiker een individueel artikel openen en lezen.

De website heeft een simpel en duidelijk ontwerp zodat bezoekers direct begrijpen wat ze kunnen doen op de site. Knoppen en navigatie zijn duidelijk zichtbaar en maken gebruik van feedforward zodat een gebruiker vooraf begrijpt wat er gebeurt na een actie.

## Gebruik
Bezoekers kunnen via het menu in de header navigeren tussen vier verschillende pagina’s waar nieuwsartikelen worden weergegeven.

Op de website kan een gebruiker:

door verschillende nieuwsartikelen bladeren

door verschillende catogorieën filteren

een artikel openen en volledig lezen

via het navigatiemenu tussen pagina’s wisselen

Voor mobiele schermen is een hamburger menu geïmplementeerd. Wanneer de gebruiker op de knop klikt, klapt het menu uit en worden de navigatielinks zichtbaar.

De website is opgebouwd volgens het mobile first principe met een one column layout. Hierdoor blijven artikelen goed leesbaar op kleinere schermen.

## Kenmerken
HTML

De HTML structuur is zo overzichtelijk mogelijk gehouden met zo min mogelijk onnodige code.

Er is gebruik gemaakt van:

semantische HTML elementen

een duidelijke heading structuur

logische en duidelijke elementnamen

Dit zorgt ervoor dat de code beter leesbaar en gestructureerd blijft.

CSS

Voor de styling is gebruik gemaakt van CSS waarbij de code zo overzichtelijk mogelijk is geschreven.

Belangrijke punten:

gebruik van CSS nesting voor een duidelijkere structuur

een mobile first layout

styling gebaseerd op de huisstijl van de Buurtcampus zoals kleuren, logo en typografie

JavaScript

JavaScript wordt gebruikt voor de interactie van het hamburger menu.

Er is een functie gemaakt die een eventlistener toevoegt aan de hamburger knop. Wanneer de gebruiker op de knop klikt wordt het menu open of dicht geklapt zodat de navigatie zichtbaar wordt op mobiele schermen.

## Installatie
Om dit project lokaal te gebruiken:

1. Clone de repository

2. Open de projectmap

3. Open index.html in de browser of start een lokale server.
